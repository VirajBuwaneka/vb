Health Care Premium Prediction

Developed a Health Insurance Premium Prediction model using machine learning to improve pricing accuracy and fairness. Key steps included data preprocessing, handling multicollinearity, and creating a risk score feature. Models like Linear Regression, Ridge Regression, and XGBoost were trained, with XGBoost achieving 97.82% accuracy. Error analysis revealed age-related prediction issues, prompting a segmented modeling approach. For users over 25, XGBoost reached 99% accuracy with minimal error. For those under 25, adding a Genetical Risk feature reduced extreme errors from 73% to 2%, making Linear Regression the best fit. The final solution was deployed via a Streamlit web app for real-time predictions.

Power of XGBoost

XGBoost (Extreme Gradient Boosting) is a highly efficient and scalable machine learning algorithm that falls under the category of ensemble learning methods. Ensemble learning combines multiple models (known as weak learners) to produce a more robust and accurate prediction than individual models. XGBoost specifically uses a boosting technique, where models are built sequentially, and each subsequent model improves upon the errors made by its predecessors.

Enhancing Classification Performance with Voting Classifier

A Voting Classifier is an ensemble learning method in machine learning that combines the predictions from multiple models (classifiers) to make a final prediction. It works by aggregating the outputs of individual models, aiming to improve the overall accuracy and robustness compared to using a single model.

K-Means Clustering

Unsupervised learning is a fundamental branch of machine learning where algorithms learn patterns from data without labeled responses. One of the most widely used unsupervised learning techniques is clustering, which groups similar data points based on certain characteristics. Among clustering algorithms, K-Means is particularly popular due to its simplicity and efficiency. It is applied in areas such as customer segmentation, anomaly detection, and image compression.
